### nimbits.com

## The nimbits project has been rebooted!

Nimbits was an open source project founded in 2001 and has gone through many iterations over the last two decades.  It was originally a data aquasition system meant for process control and automation and evolved into an IoT API.  It has been a lot of fun for me personally to develop the platform and it made me the software engineer I am today. 

Last year I gave a talk and the [Central PA Open Source Conference](https://cposc.org/) about this project and was inspired to reboot it.  

The original concept has been rebuilt into a new project called [Krill](https://krillswarm.com/) and is built from the ground up in Kotlin Multiplatform with Krill Server and
apps available for every platform. Release the Kraken!
